# Oshabz Admin Panel

This is admin panel dashboard was built with reactjs

### Tools
* Reactjs
* Material UI Icons
* React Context API
* CSS

## Features

* Responsiveness
* Lightweight

## How to run Oshabz Admin Panel
 ` git clone https://github.com/chuksdsilent/react-admin.git `

 `
 cd into the directory
 `

 `
 npm install or yarn add
 `

 `
 npm run start
 `
